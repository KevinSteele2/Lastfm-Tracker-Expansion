## The last.fm Full Album Count Tracker

last.fm is an application that tracks each song you listen to. The application itself has lots of metrics to view how many times you've listened to a song or an album or an artist, but specifically one metric I've always wanted is the number of times I've listened to an album in full. That is what this is. 

A visual display of how many times a last.fm user has listened to an album in full. It will display the albums in order of how many times they have been listened to, with a border outlining the album cover to signify the number of times each track on the album has been listened to (e.g. silver for 5, gold for 10, blue for 20, etc). You input a last.fm username, and it hits the API concurrently to fetch all the music ever listened to by the user, and groups each song together by album.
