Spotify Track Search

Overview

A music discovery web app designed for people who are hard of hearing. Built from direct user research, the app lets users filter tracks by five audio attributes that matter most to listeners with hearing differences.

Team

Kevin Steele, Cean Quinn, Lucy Acheson, William Jurewitz, Cooper Myton

The Five Filters

  • Danceability
  • Speechiness
  • Energy
  • Acousticness
  • Valence

Features

  • Slider controls for each audio attribute
  • Spotify API + Kaggle dataset integration
  • ASL translation YouTube search for each result

My Contributions

Search filtering logic, slider UI controls, and user experience testing.

Tech Stack

  • Python
  • Spotify API
  • Kaggle Songs Dataset
  • HTML
