SQ3R Reading Assistant

Overview

The SQ3R Reading Assistant was designed to utilize the SQ3R study method: Survey, Question, Read, Recite, Review. Built to support neurodivergent students by making reading and studying more engaging and accessible.

Team

Kevin Steele, Cean Quinn, Cooper Myton, Eric Jeffcoat, Jamie Whiting

Features

  • Interactive PDF viewer
  • Highlighting tool for marking important sections
  • Note-taker with local and database saving
  • Flashcard quizzer for vocabulary and definitions
  • User profiles storing notes, annotations, and PDFs
  • Built-in SQ3R prompt guiding each step of the method

My Contributions

PDF viewer implementation, overall application architecture, and highlighting feature.

Tech Stack

  • Python
  • PyMuPDF
  • Docker
